Print Statements

On this page you will find questions 1 - 7 based on print statements. The questions get gradually more difficult. Each questions has the expected output.

Using print statements write Hello World

Expected Output:

Hello World


Question 2:

Print Hello World, twice on separate lines

Expected Output:

Hello World
