Variables

On this page you will find questions 1 - 7 based on Python Variables. The questions get gradually more difficult. Each questions has the expected output.

Question 1:

Given:
int = 2
float = 2.1
str = 'numbers'
bool = True

Print the float.

Expected Output:

2.1


Question 2:

What type of variables are:
x = 3
y = 7.84
z = "nine"
k = False

Expected Output:

class 'int'
class 'float'
class 'str'
class 'bool'


Question 3:

Given:
x = 2
y = 7
z = 12
Print expected output using the given variables

Expected Output:

The variables x,y and z are: 2 7 12


Question 4:

Find the area of a sphere with radius 6
A = 4πr^2

Expected Output:

452.16


Question 5:

Find the volume of a sphere with radius 6
Round your answer to 2 decimal places
V = 4/3πr^3

Expected Output:
